

**1. Install Required Libraries:**
   - Open a command line or terminal.
   - Install the required libraries by running the following commands:
     ```
     pip install numpy pandas Pillow skyfield
     ```

**2. Open the Script and Adjust Paths:**
   - Open the Python script "hackathon1"  in a text editor or an Integrated Development Environment (IDE) of your choice.

**3. Adjust Output Directory Path:**
   - Find the line in the script where the output directory path is specified (`output_dir = r'C:\Users\hayth\OneDrive\Desktop\hackathon'`).
   - Change this path to the desired directory where you want the data to be saved.

**4. Run the Script:**
   - Save the changes to the script.
   - Open a command line or terminal.
   - Navigate to the directory where your Python script is located.
   - Run the script 

**5. Enter Data:**
   - Follow the prompts to enter the data you want to encrypt when prompted by the script.

**6. View Output:**
   - Once the script finishes running, you can find the encrypted data in an Excel file (`encrypted_data.xlsx`) saved in the directory you specified.
   - Additionally, a random image used for noise generation will be saved in the same directory.

**Note:** Ensure you have appropriate permissions to write to the specified output directory. Adjust file paths and directory locations in the script as necessary to match your system setup.
